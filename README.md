# Rock Paper Scissors Game

**Frontend Mini Project** | HTML, CSS, JavaScript

---

## Project Overview

Rock Paper Scissors is a classic hand game recreated using HTML5, CSS3, and JavaScript.
In this project, the player competes against a computer that selects moves using a random number generator.
The application displays the result of each round, updates both scores in real-time, and handles draw scenarios gracefully.

The UI is fully responsive, ensuring a smooth gaming experience across different screen sizes.
---

## Key Features

- Interactive player vs computer gameplay
- Computer move generation using Math.random()
- Real-time score tracking for both player and computer
- Clear display of win, lose, and draw outcomes
- Fully responsive layout for mobile, tablet, and desktop
- Clean UI for intuitive gameplay 

---

## Technical Highlights

- **HTML5** – SStructured layout for choices, scoreboard, and result messages  
- **CSS3** – Fully responsive design, animations, and visual cues
- **JavaScript** – Game logic, event listeners, random choice generation, and score updates 
- **Modular code structure** – Separate functions for generating moves, comparing results, and updating UI  
- **Responsive handling** – Flexbox/Grid for layout adaptability  

---

## Usage

1. Open `index.html` in your browser
2. Select Rock, Paper, or Scissors
3. Computer automatically picks a choice
4. See the round result instantly (Win / Lose / Draw)
5.Scoreboard updates live after every round
6.Continue playing as long as you want
